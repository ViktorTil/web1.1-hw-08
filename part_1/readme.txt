Пакет для загузки коллекций authors и quotes в БД (локальную, с облачной проблемы-
ошибка сертификата вылетает - писал в slack - Teacher) через контейнер:

main.py - модуль запуска - формирует и заполняет базу данных из json и обрабатывает
скрипты запросов в бесконечном цикле

models.py - модуль с моделями json-файлов для БД authors и quotes.
connect.py - модуль соединения к БД
create.py - модуль для создания и заполнения БД из json-файлов(вызывается в main).

authors.json, quotes.json - коллекции для БД.
