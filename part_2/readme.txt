Пакет для формирования и рассылки сообщений (email) из БД Contacts-
(фейковые-10 контактов):
producer.py - формирования списка для отправки и отправка в сообщениы в очередь.
consumer.py - обработка сообщений из очереди и их отправка по адресам из БД.

models.py - модель БД Contacts
connect.py - служебный файл для подсоединения к БД
create.py - модуль для создания фейковых данных(вызывается из модуля: producer.py )
